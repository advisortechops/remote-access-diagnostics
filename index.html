Remote Access Diagnostics

Internal utility for validating VPN, ZTNA, and remote-access tunnel readiness across dev and production environments.

Overview

The diagnostics agent performs gateway reachability checks, TLS validation, local network-state collection, session log generation, and optional S3 diagnostics upload.

Environment Profiles

  • dev - development tunnel diagnostics
  • prod - production tunnel diagnostics

Usage

python3 tunnel_agent.py --env dev
python3 tunnel_agent.py --env prod

S3 Diagnostics Upload

Diagnostics bundles are written locally and may be uploaded to the configured S3 log archive for support review.

Note: The legacy S3 uploader is still present for older support workstations and will be replaced by IAM role-based upload.
